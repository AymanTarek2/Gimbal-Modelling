\section*{Inverse Orientation Kinematics}
For this 3-DOF gimbal, the actuators $q_1, q_2, q_3$ directly control the orientation ($\theta_x, \theta_y, \theta_z$) of the end effector. The position ($x, y, z$) is a dependent variable fixed to a spherical surface.
Therefore, Inverse Kinematics is defined as solving the non-linear orientation system:
